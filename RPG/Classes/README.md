# Klassen-System für Skyblock

Dieser Ordner enthält alle Klassen-Definitionen und Skill-Trees.

## Klassen

Inspiriert von Wynncraft und Hypixel Skyblock:

- **Warrior**: Nahkampf-Klasse mit hoher Verteidigung
- **Mage**: Magier mit mächtigen Zaubersprüchen
- **Archer**: Fernkampf-Klasse mit Bögen
- **Assassin**: Schnelle Klasse mit hohem Schaden
- **Tank**: Defensive Klasse zum Schutz der Gruppe

## Skill-Trees

Jede Klasse hat einen Skill-Tree mit:
- Passive Fähigkeiten
- Aktive Skills
- Ultimate Fähigkeiten

## Progression

- Level 1-100
- Skill-Punkte beim Level-Up
- Verschiedene Skill-Pfade pro Klasse
