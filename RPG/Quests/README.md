# Quests für RPG

Dieser Ordner enthält alle Quest-Definitionen für den RPG Server.

## Quest-Kategorien

- **Main/**: Hauptquest-Linie
- **Side/**: Nebenquests
- **Daily/**: Tägliche Quests
- **Weekly/**: Wöchentliche Quests
- **Events/**: Event-spezifische Quests

## Quest-System

Quests folgen dem Konzept von Wynncraft mit:
- Story-getriebene Quests
- NPCs mit Dialogen
- Mehrere Objectives
- Belohnungen (Items, EXP, Geld)

## Naming Convention

Quests sollten nach folgendem Schema benannt werden:
- `KATEGORIE_NAME` (z.B. `MAIN_01_THE_BEGINNING`)
- Nummerierung für Quest-Reihenfolge
