# RPG Server - MMO Konfiguration

Dieser Ordner enthält alle Konfigurationsdateien für den RPG Server.

## Struktur

- **MythicMobs/**: MythicMobs Plugin Konfigurationen
  - **Items/**: Custom Items (Waffen, Rüstungen, Tools, etc.)
  - **Mobs/**: Custom Mobs und Bosse
  - **Skills/**: Custom Skills und Fähigkeiten
- **Quests/**: Quest-Definitionen und Quest-Linien
- **Classes/**: Klassen- und Skill-Tree Konfigurationen
- **Dungeons/**: Dungeon-Konfigurationen und Belohnungen
- **NPCs/**: NPC-Definitionen und Dialoge

## Konzept

Der RPG Server kombiniert Elemente von:
- Hypixel Skyblock (Progression, Items, Stats)
- Wynncraft RPG (Quests, Klassen, Story)

Mit zusätzlichen eigenen Verbesserungen und Features.
