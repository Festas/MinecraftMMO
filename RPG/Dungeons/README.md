# Dungeons für RPG

Dieser Ordner enthält alle Dungeon-Konfigurationen.

## Dungeon-System

Inspiriert von Hypixel Skyblock Dungeons:

- Instanzierte Dungeons für Gruppen
- Verschiedene Schwierigkeitsgrade
- Boss-Kämpfe
- Besondere Belohnungen
- Dungeon-spezifische Items

## Dungeon-Kategorien

- **Beginner/**: Dungeons für Level 1-25
- **Intermediate/**: Dungeons für Level 25-50
- **Advanced/**: Dungeons für Level 50-75
- **Expert/**: Dungeons für Level 75-100
- **Mythic/**: End-Game Dungeons für Max-Level Spieler

## Dungeon-Struktur

Jeder Dungeon enthält:
- Mob-Spawns
- Boss-Encounters
- Loot-Tables
- Schwierigkeitsgrade
