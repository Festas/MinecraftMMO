# MythicMobs Skills für Skyblock

Dieser Ordner enthält alle Custom Skill-Definitionen für Mobs und Items.

## Skill-Kategorien

- **Offensive/**: Angriffs-Skills
- **Defensive/**: Verteidigungs-Skills
- **Support/**: Unterstützungs-Skills
- **Special/**: Spezielle Effekte und Mechaniken

## Naming Convention

Skills sollten beschreibende Namen haben:
- `FireballBarrage` - Feuert mehrere Feuerbälle
- `HealingAura` - Heilung in einem Bereich
- `SummonMinions` - Beschwört Helfer
