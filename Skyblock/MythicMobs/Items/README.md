# MythicMobs Items für Skyblock

Dieser Ordner enthält alle Custom Item-Definitionen für den Skyblock Server.

## Item-Kategorien

- **Weapons/**: Waffen (Schwerter, Bögen, Stäbe, etc.)
- **Armor/**: Rüstungsteile
- **Tools/**: Werkzeuge (Spitzhacken, Äxte, etc.)
- **Accessories/**: Accessoires und Talismane
- **Materials/**: Crafting-Materialien und Ressourcen
- **Consumables/**: Verbrauchsgegenstände (Tränke, Nahrung, etc.)

## Naming Convention

Items sollten nach folgendem Schema benannt werden:
- `KATEGORIE_NAME_TIER` (z.B. `SWORD_FIRE_LEGENDARY`)
- Tier: COMMON, UNCOMMON, RARE, EPIC, LEGENDARY, MYTHIC
