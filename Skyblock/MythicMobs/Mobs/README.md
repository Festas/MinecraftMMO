# MythicMobs Mobs für Skyblock

Dieser Ordner enthält alle Custom Mob-Definitionen für den Skyblock Server.

## Mob-Kategorien

- **Bosses/**: Boss-Mobs für Dungeons und Events
- **Elite/**: Elite-Mobs mit besonderen Fähigkeiten
- **Common/**: Normale feindliche Mobs
- **Passive/**: Passive oder neutrale Mobs

## Naming Convention

Mobs sollten nach folgendem Schema benannt werden:
- `MOB_NAME_LEVEL` (z.B. `ZOMBIE_WARRIOR_50`)
- Level bestimmt die Schwierigkeit
