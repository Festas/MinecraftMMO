# Progression-System - Skyblock-Server

Skyblock-spezifische Progression und Skill-Integration.

---

## Progression-Pfade

### 1. Island-Progression
- Island-Level steigern
- Upgrades freischalten
- Prestige erreichen

### 2. MMO-Progression
- Klassen-Level (1-100)
- Skills freischalten
- Ausrüstung verbessern

### 3. Collection-Progression
- Ressourcen sammeln
- Rezepte freischalten
- Collection-Milestones
