# Item-System - Übersicht

Dokumentation des umfassenden Item-Systems mit Oraxen, MMOItems und MythicCrucible.

---

## Item-Pipeline

**Workflow:** Oraxen (Textur) → MMOItems (Stats) → MythicCrucible (Crafting)

1. **Oraxen** - Custom Texturen und Models
2. **MMOItems** - Stats, Abilities, Tiers
3. **MythicCrucible** - Crafting-Rezepte

---

## Tier-System

Das Spiel nutzt **7 Item-Tiers:**

1. **Gewöhnlich** (Common) - Grau
2. **Ungewöhnlich** (Uncommon) - Grün
3. **Selten** (Rare) - Blau
4. **Episch** (Epic) - Lila
5. **Legendär** (Legendary) - Orange
6. **Mythisch** (Mythic) - Rot
7. **Göttlich** (Divine) - Gold _(geplant)_
