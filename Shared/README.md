# Shared Resources

Dieser Ordner enthält gemeinsame Ressourcen, die von mehreren Servern genutzt werden.

## Inhalte

- **Config/**: Gemeinsame Konfigurationsdateien
- **Resources/**: Texturpacks, Sounds, etc.

## Verwendung

Diese Dateien können von allen Servern referenziert werden, um Duplikation zu vermeiden.
